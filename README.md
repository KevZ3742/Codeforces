### **Extensions & Tools**

#### Browser Extensions
*   [**Competitive Companion**](https://github.com/jmerle/competitive-companion) – Parses problems directly from Codeforces into your editor.
*   [**CPH Submit**](https://github.com/agrawal-d/cph-submit) – Automatically submits solutions from VS Code.

#### VS Code Extensions
*   [**Competitive Programming Helper (cph)**](https://marketplace.visualstudio.com/items?itemName=DivyanshuAgrawal.competitive-programming-helper) – Tool for running test cases locally.
